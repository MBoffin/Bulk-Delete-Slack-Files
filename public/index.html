Delete Slack Files in Bulk

Slack allows for a maximum of 100 files to be deleted at one go. If your files are more than 100, refresh and repeat untill they are all deleted.

Instructions

  1. Sign into Slack.
  2. Go to this page: https://api.slack.com/custom-integrations/legacy-tokens
  3. Scroll down and click the big, green Request Token button next to the PIGSquad team.
  4. Copy the token it creates. (It will look crazy, like abcd-1234567890-1234567890-12345678901-ab1cd2ef3g.)
  5. Paste your Slack token into this textbox:
  6. Select your name:
  7. Delete files older than: (Please select now!)
  8. Delete files
